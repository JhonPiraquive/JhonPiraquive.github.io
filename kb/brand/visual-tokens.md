# Visual tokens

## Paleta
| Token | Hex | Uso |
|-------|-----|-----|
| `--color-primary` | `#0A2540` | Fondos hero, confianza, liderazgo |
| `--color-secondary` | `#00C2FF` | IA, innovación, cloud |
| `--color-accent` | `#6B4EFF` | Creatividad, visión |
| `--color-neutral-white` | `#FFFFFF` | Superficies |
| `--color-neutral-light` | `#F4F6F8` | Fondos secciones |
| `--color-neutral-dark` | `#1E293B` | Texto cuerpo |

## Tipografías
- **Montserrat** (600–700): títulos, logo
- **Inter** (400–500): cuerpo, docs

## Claymorphism
- `border-radius`: 20–28px
- Sombra outer suave azul + inner highlight blanco
- Bordes sutiles con secondary/accent en hover

## Do / Don't
- Do: espaciado generoso, gradientes suaves, profundidad moderada
- Don't: neón, glass extremo, iconografía gaming, gradientes cripto
