# Brief — Clase 3: Sistemas operativos

## Objetivos de aprendizaje

1. Definir sistema operativo y explicar el rol del planificador de procesos.
2. Clasificar SO por entorno (escritorio, móvil, servidor, tiempo real) y comparar Windows, macOS, Linux, Android, iOS.
3. Describir instalación y configuración inicial (requisitos, BIOS/UEFI, particiones, drivers).
4. Usar consola Linux y equivalentes Windows; distinguir rutas absolutas y relativas.
5. Gestionar usuarios, grupos y permisos rwx (useradd, passwd, chmod).
6. Explicar niveles de respaldo 1–4 (espejo, NAS, cloud, geo-redundante).
