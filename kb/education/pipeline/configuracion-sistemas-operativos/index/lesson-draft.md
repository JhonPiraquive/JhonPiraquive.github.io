# Lesson draft — Índice

## Bienvenida

Curso de configuración de sistemas operativos sobre hardware real: arquitectura, almacenamiento, periféricos, instalación y consola.

## Calendario

| Clase | Tema | Slug |
|-------|------|------|
| 1 | Arquitectura del computador | clase-01-arquitectura-computador |
| 2 | Almacenamiento y periféricos | clase-02-dispositivos-almacenamiento |
| 3 | Sistemas operativos | clase-03-sistemas-operativos |

## Prerrequisitos

Uso básico de PC; opcional POSW modelo cliente-servidor.

## Navegación

Cada clase: hub + páginas ~15–20 min. Clase 1: 5 páginas; Clase 2: 4; Clase 3: 6.

## Objetivos del curso

Competencia: instalar y configurar SO documentando hardware. Resultados: arquitectura, almacenamiento, inventario, instalación, consola, usuarios y respaldo.

## Cierre hub

Al completar: especificar hardware, elegir almacenamiento, instalar SO, gestionar usuarios y respaldos.
