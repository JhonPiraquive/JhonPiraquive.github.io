# Introducción al curso — Configuración de Servicios Web

## Competencia del curso

Instalar y configurar artefactos de software relacionados con protocolos y servicios de Internet.

## Duración

4 clases × 2 horas = **8 horas** de aula (ajustable a 6 h omitiendo profundización opcional).

## Prerrequisitos

- POSW: servicios web, protocolos de seguridad, modelo cliente-servidor
- Conocimiento básico de HTTP y navegador

## Calendario sugerido

| Clase | Tema | Duración |
|-------|------|----------|
| 1 | Fundamentos web: navegadores, IP, dominios, DNS | 2 h |
| 2 | Hosting, correo corporativo, HTTPS/TLS | 2 h |
| 3 | Nube, SSH, SFTP, administración remota | 2 h |
| 4 | Docker, VMs, diagnóstico y reto integrador | 2 h |

## Resultados de aprendizaje (curso completo)

Al terminar las 4 clases el estudiante podrá:

1. Configurar y diagnosticar navegadores como cliente web.
2. Explicar IP pública/privada y configurar registros DNS en un dominio .co o .com.
3. Elegir hosting, desplegar sitio con HTTPS y correo corporativo (MX, SPF, DKIM).
4. Administrar servidores por SSH/SFTP y paneles remotos.
5. Crear entornos Docker y VM para pruebas.
6. Resolver fallos típicos de DNS, TLS, correo y contenedores.

## Cómo usar este material

Cada clase tiene objetivos propios, teoría extendida, ejercicios interactivos, guía de tiempo para el docente y mini-quiz. Avanza en orden: Clase 1 → 4.
