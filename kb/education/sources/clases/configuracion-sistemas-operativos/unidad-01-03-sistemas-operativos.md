# Unidades 1–3 — Configuración de Sistemas Operativos

Material fuente para el track `configuracion-sistemas-operativos`. Contenido publicado en TSX bajo `src/components/teaching/lessons/configuracion-sistemas-operativos/`.

## Unidad 1 — Arquitectura de un computador

- Arquitectura conceptual (CPU, buses, memoria, E/S)
- Chasis del computador
- CPU: CU, ALU, funciones
- Microprocesador: núcleos, hilos, caché L1/L2/L3, bus de datos, velocidad de reloj (GHz), refrigeración
- Memoria RAM: DRAM, SRAM, DDR/DDR2/DDR3/DDR4
- Memoria ROM, BIOS, EEPROM/FLASH

## Unidad 2 — Dispositivos de almacenamiento

- Discos HDD, SSD, SAS; disco óptico
- Periféricos (monitor: resolución, píxeles)
- Hoja de vida del PC
- Licencias de software (libre, propietario, dominio público, freeware, shareware, privativo, comercial)

## Unidad 3 — Sistemas operativos

- Clasificación SO (escritorio, móvil, tiempo real)
- Windows, macOS, Linux
- Instalación y configuración
- Consola de comandos (Linux y homólogos Windows)
- Rutas absolutas y relativas
- Usuarios, root, grupos, permisos (chmod)
- Protocolos de respaldo (niveles 1–4)
