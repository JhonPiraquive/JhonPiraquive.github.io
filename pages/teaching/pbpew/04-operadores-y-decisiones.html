Operadores y control de flujo: if / switch

Aritmética, comparación y lógica; decisiones con if, else if y switch.

Leer

Objetivos

  • Usar operadores aritméticos, de comparación y lógicos.
  • Escribir ramas con if, else if y else.
  • Elegir entre varias opciones con switch.

Operadores aritméticos

let a = 10 + 3; // 13
let b = 10 - 3;
let c = 10 * 3;
let d = 10 / 3;
let e = 10 % 3; // resto: 1
let f = 2 ** 8; // potencia: 256

Comparación

Usa === y !== para comparar valor y tipo. == hace coerción y suele confundir.

5 === "5"; // false
5 == "5"; // true (coerción)
10 > 3; // true
10 <= 10; // true

Operadores lógicos

true && false; // AND → false
true || false; // OR → true
!true; // NOT → false

// Cortocircuito útil:
const nombre = "" || "invitado"; // "invitado"

if, else if, else

const nota = Number(prompt("Nota (0-5):"));

if (Number.isNaN(nota)) {
  console.log("No escribiste un número válido");
} else if (nota >= 3.0) {
  console.log("Aprobado");
} else {
  console.log("Debes mejorar");
}

switch

const dia = "lunes";

switch (dia) {
  case "lunes":
  case "miércoles":
    console.log("Hay clase de PBPEW");
    break;
  case "viernes":
    console.log("Repaso");
    break;
  default:
    console.log("Otro día");
}
flowchart TD A[Condición] -->|true| B[Bloque A] A -->|false| C{¿Otra condición?} C -->|true| D[Bloque B] C -->|false| E[Bloque por defecto]

Resumen

  • Prefiere === para igualdad estricta.
  • if encadena decisiones; switch lista valores discretos.
  • No olvides break en switch si no quieres “caída” al siguiente caso.
