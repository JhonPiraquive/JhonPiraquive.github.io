Bucles: for, while, do-while · break, continue · try/catch

Repetir acciones con control y manejar excepciones sin romper toda la aplicación.

Leer

Bucle for

for (let i = 0; i < 3; i++) {
  console.log("iteración", i);
}

Bucle while

let n = 0;
while (n < 3) {
  console.log(n);
  n++;
}

Bucle do...while

Garantiza al menos una ejecución antes de comprobar la condición.

let k = 0;
do {
  console.log("al menos una vez", k);
  k++;
} while (k < 0);

break y continue

for (let i = 0; i < 5; i++) {
  if (i === 2) continue; // salta a la siguiente iteración
  if (i === 4) break; // sale del bucle
  console.log(i);
}
// Imprime 0, 1, 3

try / catch / finally

Permite atrapar errores en tiempo de ejecución y reaccionar (log, valor por defecto) en lugar de detener todo el script.

function dividir(a, b) {
  if (b === 0) {
    throw new Error("División por cero");
  }
  return a / b;
}

try {
  console.log(dividir(4, 0));
} catch (err) {
  console.error("No se pudo dividir:", err.message);
} finally {
  console.log("Esto se ejecuta siempre");
}

Resumen

  • for es ideal cuando conoces el rango; while cuando la condición es más libre.
  • break sale del bucle; continue salta al siguiente paso.
  • try/catch rodea código que puede fallar y te permite recuperarte.
