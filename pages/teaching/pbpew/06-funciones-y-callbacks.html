Tema 06

Funciones, return, parámetros y callbacks

Empaquetar lógica reutilizable y pasar funciones como valores.

Leer

Declaración de función

function saludar(nombre) {
  return `Hola, ${nombre}`;
}

console.log(saludar("Patricia"));

Función como expresión (anónima)

const duplicar = function (x) {
  return x * 2;
};

const triple = (x) => x * 3; // flecha, retorno implícito

Parámetros y argumentos

Los parámetros son los nombres en la definición; los argumentos son los valores que pasas en la llamada.

function sumar(a, b) {
  return a + b;
}

sumar(2, 5); // argumentos 2 y 5

return

Devuelve un valor al llamador y termina la función en ese punto.

Callbacks

Un callback es una función que pasas a otra función para que la ejecute más tarde (por ejemplo, cuando ocurre un evento).

function repetir(n, fn) {
  for (let i = 0; i < n; i++) {
    fn(i);
  }
}

repetir(3, (i) => console.log("paso", i));

Resumen

  • Las funciones agrupan lógica y permiten reutilización y pruebas aisladas.
  • return envía el resultado hacia fuera.
  • Los callbacks conectan tu código con APIs que “llamarán de vuelta” (eventos, temporizadores, promesas).
