Arrays

const nums = [2, 4, 6];
nums.push(8); // añade al final
const ultimo = nums.pop();

const dobles = nums.map((x) => x * 2);
const pares = nums.filter((x) => x % 2 === 0);
const suma = nums.reduce((acc, x) => acc + x, 0);

Métodos comunes

  • map: transforma cada elemento → nuevo array.
  • filter: se queda con elementos que cumplen condición.
  • reduce: combina en un solo valor (suma, conteo, objeto).
  • find / some / every: búsqueda y comprobaciones.
  • forEach: ejecuta efecto por elemento sin devolver array nuevo.

JSON

JSON (JavaScript Object Notation) es un formato de texto para intercambiar datos: subconjunto de la sintaxis literal de objetos/arrays de JS.

const curso = {
  nombre: "PBPEW",
  horas: 40,
  activo: true,
};

const texto = JSON.stringify(curso);
console.log(texto);

const otraVez = JSON.parse(texto);
console.log(otraVez.nombre);

Objetos literales

const alumno = {
  id: 42,
  nombre: "Sofía",
  presentarse() {
    return `Soy ${this.nombre}`;
  },
};

console.log(alumno.presentarse());

Resumen

  • Los arrays son listas indexadas; los métodos de orden superior simplifican patrones repetidos.
  • JSON serializa objetos a texto para APIs y almacenamiento.
  • Los objetos agrupan datos y funciones relacionadas.
