this, ámbito (scope), clases y herencia

De dónde “salen” las variables y cómo las clases modelan objetos con herencia.

Leer

Ámbito (scope)

  • Global: declaraciones fuera de funciones (evita contaminar el global en apps grandes).
  • De función: variables var locales a la función.
  • De bloque: let y const dentro de {}.

this

El valor de this depende de cómo se llama la función. En métodos de objeto, suele ser el objeto. En funciones flecha, this se hereda del ámbito donde se definió.

const caja = {
  valor: 10,
  mostrar() {
    console.log(this.valor);
  },
};
caja.mostrar(); // 10

Clases y métodos

class Rectangulo {
  constructor(ancho, alto) {
    this.ancho = ancho;
    this.alto = alto;
  }

  area() {
    return this.ancho * this.alto;
  }
}

const r = new Rectangulo(4, 5);
console.log(r.area()); // 20

Herencia

class Cuadrado extends Rectangulo {
  constructor(lado) {
    super(lado, lado);
  }
}

const c = new Cuadrado(3);
console.log(c.area()); // 9

Resumen

  • El ámbito limita dónde existe un nombre.
  • this enlaza el contexto; presta atención al modo de llamada.
  • Las clases son azúcar sintáctico sobre prototipos; extends y super expresan herencia.
