Map, Set, pila (LIFO) y cola (FIFO)

Estructuras integradas en el lenguaje y dos patrones clásicos con arrays.

Leer

Map

Pares clave-valor con cualquier tipo de clave (no solo strings como en objetos).

const edades = new Map();
edades.set("Ana", 21);
edades.set("Luis", 22);
console.log(edades.get("Ana"));

Set

Conjunto de valores únicos.

const ids = new Set([1, 2, 2, 3]);
console.log(ids.size); // 3

Pila (LIFO: Last In, First Out)

Como una pila de platos: el último que entra es el primero que sale. Con array: push + pop.

flowchart LR subgraph pila [Pila] A[push 1] B[push 2] C[pop devuelve 2] end 
const pila = [];
pila.push("A");
pila.push("B");
console.log(pila.pop()); // "B"

Cola (FIFO: First In, First Out)

Como una fila en el banco: el primero en llegar es el primero en ser atendido. Con array: push al final + shift al inicio (u optimizaciones en producción).

const cola = [];
cola.push("ticket1");
cola.push("ticket2");
console.log(cola.shift()); // "ticket1"

Resumen

  • Map y Set son estructuras integradas muy usadas en JS moderno.
  • Pila y cola son patrones; implementación típica con arrays y métodos adecuados.
