Manipulación del DOM y eventos

querySelector, cambiar contenido y escuchar clics, teclado y envío de formularios.

Leer

Seleccionar nodos

const titulo = document.querySelector("h1");
const botones = document.querySelectorAll("button.item");
const porId = document.getElementById("app");
const porClase = document.getElementsByClassName("card");

Cambiar contenido y estilos

const p = document.querySelector("#mensaje");
p.textContent = "Nuevo texto";
p.innerHTML = "<strong>Importante</strong>";
p.style.color = "#0f766e";

Crear y eliminar nodos

const lista = document.querySelector("#lista");
const li = document.createElement("li");
li.textContent = "Elemento nuevo";
lista.appendChild(li);
lista.removeChild(li);

Eventos

const btn = document.querySelector("#ok");

btn.addEventListener("click", (evento) => {
  console.log("clic", evento.target);
});

document.addEventListener("keydown", (e) => {
  if (e.key === "Enter") console.log("Enter");
});
sequenceDiagram participant U as Usuario participant N as Navegador participant L as Listener JS U->>N: click en botón N->>L: dispara evento L->>N: actualiza DOM

Delegación de eventos: escucha en un contenedor y usa event.target para saber qué hijo originó el evento — útil con listas largas.

Resumen

  • El DOM es mutable: crear, leer, actualizar y borrar nodos.
  • addEventListener separa HTML de comportamiento.
  • Los objetos de evento transportan detalles (tecla, botón, objetivo).
