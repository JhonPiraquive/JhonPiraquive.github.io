Material para estudiantes universitarios que empiezan en programación: conceptos claros, ejemplos en código, diagramas y enlaces entre temas.

Ver temas

Objetivos de aprendizaje

  • Entender qué es JavaScript, dónde se ejecuta y cómo encaja el DOM en una página web.
  • Escribir scripts en HTML, usar la consola del navegador y manejar variables, tipos y control de flujo.
  • Usar funciones, arrays, objetos, JSON y clases; conocer asincronía y fetch.
  • Manipular el DOM, responder a eventos y leer código asíncrono con promesas y async/await.

Prerrequisitos

  • Saber crear un archivo .html y abrirlo en el navegador.
  • Conceptos básicos de HTML (etiquetas, atributos). No se requiere otro lenguaje de programación.

Estructura del curso (mapa)

Orden recomendado: del 01 al 12. Cada tema incluye objetivos, ejemplos y un bloque para practicar.

01Introducción a JS y el DOM 02JavaScript en HTML y consola 03Variables, hoisting y tipos 04Operadores y decisiones 05Bucles y manejo de errores 06Funciones y callbacks 07Arrays, JSON y objetos 08this, ámbito, clases y herencia 09Estructuras de datos (Map, Set, pila, cola) 10DOM y eventos 11Asincronía y plantillas 12AJAX y fetch

Cómo usar las herramientas de desarrollo

Consola: en Chrome o Edge, pulsa F12 o Ctrl+Shift+J (Linux/Windows) y abre la pestaña “Console”. Ahí puedes escribir JavaScript y ver mensajes de console.log.

Fuentes (Sources): para depurar con puntos de interrupción. Elements: para inspeccionar el DOM en vivo.
