DataLoader pattern: Los resolvers de GraphQL pueden generar el problema N+1 (N queries para N items). DataLoader lo resuelve agrupando y deduplicando las consultas en una sola batch.

Patrones de Arquitectura de APIs

API Gateway

Punto de entrada Гәnico que centraliza auth, rate limiting, logging y routing a microservicios.

BFF (Backend for Frontend)

Una API especГӯfica por tipo de cliente (BFF Mobile, BFF Web). Evita que el frontend cargue datos innecesarios.

Strangler Fig

PatrГіn de migraciГіn donde se reemplaza gradualmente un monolito aГұadiendo nuevos endpoints en microservicios, mientras los viejos siguen funcionando.

CQRS

Command Query Responsibility Segregation: separar las operaciones de lectura (Queries) de las de escritura (Commands) en modelos y bases de datos distintos.
